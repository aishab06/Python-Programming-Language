
# Python Programming Language



A collection of daily Python practice notebooks covering core language fundamentals — built while learning Python with the goal of becoming a backend/automation-focused developer.

## What's in here

| Notebook | Topics covered |
|---|---|
| `Python_String_24June.ipynb` | String manipulation, slicing, formatting, common string methods |
| `Python_List_25June.ipynb` | List operations, indexing, comprehensions, common list methods |
| `Python_Dictionary_26June.ipynb` | Dictionaries, key-value operations, iteration, sorting by key/value |
| `Tuesday_Work_23June.ipynb` | General practice exercises |

Each notebook contains worked examples and problems solved step-by-step, with an emphasis on understanding the *why* behind each solution, not just memorizing syntax.

## Why this repo exists

I'm working toward becoming a Python developer, building from fundamentals up. This repo tracks that progress — daily practice covering data structures, control flow, and problem-solving patterns that come up repeatedly in real code and technical interviews.

## Topics practiced so far

- String manipulation and built-in string methods
- List operations, comprehensions, and common patterns (deduplication, filtering, sorting)
- Dictionary operations — counting, sorting by value, key/value lookups
- Sets and tuples — operations, use cases, hashability
- Loop patterns — accumulators, nested loops, while vs. for
- Problem-solving exercises — palindrome checks, pair-sum problems, pattern printing

## Notes

This is an active learning repo — notebooks are added as new topics are covered. Code favors clarity over cleverness at this stage, with explicit loops alongside more concise one-liner alternatives where relevant, to demonstrate understanding of the underlying logic.
